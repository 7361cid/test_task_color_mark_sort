1 Запуск консольного приложения  
`python3 console_app.py`

2 Запуск Unint тестов  
`python3 tests_unit.py`

3 Проведение perfomance теста  
В одном терминале `python3 perfomance_test.py`  
Во втором терминале для оценки памяти `ps -p {pid} -o pid,rss,vsz` pid будет выведен в 1ый терминал,  
также инструкция о том когда будет нужен повторный вызов будет в терминале 1  

4 Описание тестов в файле `test_documentation.pdf`
